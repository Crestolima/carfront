# Car Rental Application

## Overview
This is the client-side application for a car rental service built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

## Installation
```bash
npm install
npm start
```

## Features
- User authentication
- Car listing and search
- Booking management
- User profile
- Responsive design
